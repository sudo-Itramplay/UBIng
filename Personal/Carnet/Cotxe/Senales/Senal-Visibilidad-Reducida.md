# Señal de Visibilidad Reducida

## Pregunta de examen (errada)

> No conozco la señal de visibilidad reducida.

## Respuesta correcta

**VERDAD.** Hay varias señales relacionadas con visibilidad reducida. Las mas importantes son las señales de peligro por **niebla (P-15)**, **precipitaciones (P-16)** y **vent lateral (P-17)**, ademas del panel de luces obligatorias.

## Explicacion detallada

### Señales de peligro por condiciones meteorologicas

| Señal | Nombre | Significado |
|-------|--------|-------------|
| **P-15** | Niebla o bancos de niebla | Proximidad de zonas con niebla frecuente |
| **P-16** | Precipitaciones | Proximidad de zonas con lluvia, nieve o granizo frecuentes |
| **P-17** | Viento lateral | Proximidad de zonas con viento lateral fuerte |
| **P-18** | Calzada resbaladiza | Proximidad de zonas con calzada resbaladiza frecuente |

### Panel de luces obligatorias (R-413)

Cuando hay visibilidad reducida por niebla, lluvia intensa, nieve o polvo:
- Se activa el panel **R-413** (circulo azul con pictograma de luces)
- Indica que es **obligatorio** encender las luces de cruce
- Este panel suele aparecer en autopistas, autovias y tuneles

### Marcas viales en visibilidad reducida

- **Tacha reflectante**: marcas en el suelo que brillan con las luces
- **Poste de galibo**: postes laterales con reflectantes
- **Linea discontinua** mas corta: para mejor visibilidad

### Luces obligatorias con visibilidad reducida

| Visibilidad | Luces obligatorias |
|-------------|-------------------|
| Niebla ligera | Luces de cruce |
| Niebla densa | Luces de cruce + antiniebla trasera |
| Lluvia intensa | Luces de cruce + antiniebla trasera |
| Nieve | Luces de cruce + antiniebla trasera |
| Polvo | Luces de cruce + antiniebla trasera |

### Errores habituales

| Error | Peligro |
|-------|---------|
| No encender luces con niebla | No te ven otros conductores |
| Usar luces de carretera con niebla | Reflejo que reduce visibilidad |
| No respetar la señal de luces obligatorias | Multa y riesgo de accidente |
| No reducir la velocidad | Colision multiple |

## Recordatorio

**Niebla = P-15.** **Precipitaciones = P-16.** **Luces obligatorias = R-413.** Siempre encender luces de cruce.
