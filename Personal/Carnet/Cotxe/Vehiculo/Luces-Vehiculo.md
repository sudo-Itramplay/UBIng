# Luces del Vehiculo: Guia Completa

## Pregunta de examen (errada)

> ¿Que es la luz de estacionamiento? ¿Y la de posicion? ¿Como se encienden? Explica las distintas luces del coche/moto.

## Respuesta correcta

### Luces de posicion

| Aspecto | Descripcion |
|---------|-------------|
| **Funcion** | Señalar la presencia y el ancho del vehiculo |
| **Que se ilumina** | Luces delanteras y traseras de color blanco/rojo |
| **Obligatoriedad** | Cuando hay poca visibilidad (noche, niebla, tunel, etc.) |
| **No son de carretera** | Solo sirven para ser visto, no para ver |

### Luz de estacionamiento

| Aspecto | Descripcion |
|---------|-------------|
| **Funcion** | Señalar el vehiculo cuando esta parado fuera de la via (o en espacio habilitado) |
| **Que se ilumina** | Solo una luz delantera y una trasera (el lado del vehiculo hacia la calzada) |
| **Uso** | Cuando el vehiculo esta estacionado en zona sin iluminacion |
| **Diferencia con posicion** | Solo se enciende UN lateral, no los dos |

### Luz de carretera (larga distancia)

| Aspecto | Descripcion |
|---------|-------------|
| **Funcion** | Iluminar la via a gran distancia |
| **Uso** | Carreteras sin iluminacion ni trafico en sentido contrario |
| **Prohibido** | En urbano, con trafico en sentido contrario, con niebla densa |
| **Obligacion** | Adelantamiento: se apaga al cruzar |

### Luz de cruce (corta distancia)

| Aspecto | Descripcion |
|---------|-------------|
| **Funcion** | Iluminar la via sin deslumbrar a otros |
| **Uso** | Circulacion normal nocturna, dia con visibilidad reducida |
| **Obligatoriedad** | En tunel, noche, niebla, lluvia intensa |

### Luz antiniebla delantera y trasera

| Aspecto | Descripcion |
|---------|-------------|
| **Delantera** | Solo con niebla densa (poco habitual) |
| **Trasera** | Rojo, obligatoria con niebla, lluvia intensa o nieve |
| **No usar** | Con buena visibilidad (deslumbra a otros) |

### Luz de freno

| Aspecto | Descripcion |
|---------|-------------|
| **Funcion** | Indicar que el vehiculo esta frenando |
| **Activacion** | Se enciende automaticamente al pisar el freno |
| **Obligatoriedad** | Si, es un elemento de seguridad pasiva |

### Luz de marcha atras

| Aspecto | Descripcion |
|---------|-------------|
| **Funcion** | Iluminar la zona trasera y señalar que se circula marcha atras |
| **Color** | Blanca |
| **Activacion** | Al engranar la marcha atras |

### Luz de emergencia (warning)

| Aspecto | Descripcion |
|---------|-------------|
| **Funcion** | Advertir a otros usuarios de un peligro o averia |
| **Que se ilumina** | Todas las luces de intermitencia a la vez |
| **Uso** | Averia, accidente, detencion en zona peligrosa |
| **Prohibido** | Como aviso de radar, para cambiar de direccion |

## Cuadro resumen

| Luz | Color | ¿Cuando? |
|-----|-------|----------|
| Posicion | Blanco/rojo | Noche, visibilidad reducida |
| Estacionamiento | Blanco/rojo (1 lado) | Estacionado sin luz |
| Cruz | Blanco | Circulacion normal nocturna |
| Carretera | Blanco | Noche en via sin trafico |
| Frenado | Rojo | Al frenar |
| Niebla trasera | Rojo | Niebla, lluvia intensa |
| Marcha atras | Blanco | Circular hacia atras |

## Errores habituales

| Error | Peligro |
|-------|---------|
| Usar luz de carretera en urbano | Deslumbra a otros vehiculos |
| Usar antiniebla con buena visibilidad | Deslumbra, multa |
| Confundir estacionamiento con posicion | Estacionamiento solo un lado, posicion los dos |
| No usar luz de freno en retencion | Colision trasera |

## Recordatorio

**Posicion = ser visto.** **Cruz = ver sin deslumbrar.** **Carretera = maxima luz.** **Estacionamiento = un lateral solo.**
