# A2: ¿Se pueden Conducir Tractores?

## Pregunta de examen (errada)

> Con el permiso A2 se pueden conducir tractores.

## Respuesta correcta

**FALSO.** Con el permiso **A2 NO se pueden conducir tractores**. Los tractores agricolas requieren el permiso **B** o un permiso especifico para maquinaria agricola.

## Explicacion detallada

### ¿Que se puede conducir con A2?

| Vehiculo | ¿Con A2? |
|----------|----------|
| **Motocicletas** hasta 35 kW | SI |
| **Motocicletas** con relacion potencia/peso maximo 0,2 kW/kg | SI |
| **Ciclomotores** | SI |
| **Vehiculos de 3 ruedas** hasta 15 kW | SI |
| **Tractores agricolas** | **NO** |
| **Cuadriciclos** | Depende |

### ¿Por que no tractores?

- El permiso A2 esta diseñado para **motocicletas**
- Los tractores son **maquinaria agricola** o industrial
- Requieren formacion especifica para su manejo
- El permiso **B** si permite conducir algunos tractores (dependiendo de la MMA y velocidad)

### Permisos para tractores

| Tipo de tractor | Permiso requerido |
|-----------------|-------------------|
| **Tractor agricola ligero** | Permiso B (si cumple requisitos) |
| **Tractor agricola pesado** | Permiso especifico o profesional |
| **Maquinaria de obra** | Carnet de maquinista |

### Relacion con otros permisos

| Permiso | Motocicletas | Tractores |
|---------|--------------|-----------|
| **AM** | Ciclomotores | NO |
| **A1** | Motos hasta 125 cc | NO |
| **A2** | Motos hasta 35 kW | NO |
| **A** | Todas las motos | NO |
| **B** | NO | SI (algunos) |

### Errores habituales

| Error | Peligro |
|-------|---------|
| Creer que A2 incluye vehiculos agricolas | Multa por conducir sin permiso |
| Conducir un tractor con A2 | Sancion administrativa |
| Confundir cuadriciclo con tractor | Son categorias diferentes |

## Recordatorio

**A2 = motos, NO tractores.** Los tractores requieren B o permiso especifico.
