# Discrepancias entre Tests de Autoescuela y DGT

## Introduccion

Este documento recoge las discrepancias encontradas entre lo que indican algunas aplicaciones de autoescuela/tests y la normativa real de la DGT. **Cuando hay discrepancia, prevalece la DGT.**

---

## Discrepancias Generales (Carnet B)

### 1. Repartidores y cinturon de seguridad

| Fuente | Lo que dice |
|--------|-------------|
| **Tests antiguos** | Los repartidores estan exentos del cinturon **dentro de poblado** |
| **DGT (normativa vigente)** | El cinturon es **obligatorio para todos** los ocupantes en todas las vias, **sin excepcion** por tipo de trabajo |
| **Conclusion** | Los repartidores/distribuidores **NO estan exentos** nunca |
| → Ver | [[Cintura-Seguridad-Exenciones]] |

### 2. ITV — Primera inspeccion

| Fuente | Lo que dice |
|--------|-------------|
| **Tests antiguos** | Primera ITV a los **2 anos** |
| **DGT (Reglamento General de Vehiculos)** | Primera ITV a los **4 anos** desde la matriculacion |
| **Conclusion** | Turismos: 4 anos primera, luego 2-2-1 |
| → Ver | [[ITV-Primera-Inspeccion]] |

### 3. Permiso B — "Camion"

| Fuente | Lo que dice |
|--------|-------------|
| **Tests mal redactados** | "¿Puedo conducir un camion con B?" → A veces dan respuestas confusas |
| **DGT** | Con B se conduce **furgoneta (N1 < 3.500 kg MMA)**. **Camion (N2/N3) = permiso C/C1** |
| **Conclusion** | Si el test dice "camion", la respuesta es **NO**. Si dice "furgoneta", puede ser **SI** |
| → Ver | [[Permiso-B-Vehiculos-Pesados]] |

## Discrepancias Especificas de Coche (B)

### 4. Luces con niebla

| Fuente | Lo que dice |
|--------|-------------|
| **Tests antiguos** | Usar luces de carretera (largas) para ver mejor |
| **DGT** | Con niebla = **luces de cruce + antiniebla trasera**. **NUNCA** luces de carretera (se reflejan) |
| **Conclusion** | Carretera con niebla = peligroso y sancionable |
| → Ver | [[Luces-Niebla-Lluvia-Intensa]] |

### 5. Rueda de repuesto — Presion

| Fuente | Lo que dice |
|--------|-------------|
| **Tests confusos** | "Hinchar a la presion maxima" |
| **DGT / Fabricante** | Presion indicada por el **fabricante** (ej: 2,2 bar), NO la maxima del neumatico |
| **Conclusion** | Maxima = limite de seguridad. Usar la presion recomendada |
| → Ver | [[Ruedas-Repuesto-Presion]] |

### 6. Servofreno

| Fuente | Lo que dice |
|--------|-------------|
| **Tests tradicionales** | Funciona por "depresion del motor" |
| **DGT / Tecnica moderna** | En vehiculos electricos/hibridos funciona por **sistema electrico/hidraulico**, no por depresion |
| **Conclusion** | Depresion del motor = solo vehiculos de combustion |
| → Ver | [[Servofreno]] |

### 7. Cambio de sentido en glorieta

| Fuente | Lo que dice |
|--------|-------------|
| **Tests confusos** | "En glorieta no se puede cambiar de sentido" |
| **DGT** | En glorieta **SI se puede** cambiar de sentido dando la vuelta completa |
| **Conclusion** | Siempre permitido dejando el macizo a la izquierda |
| → Ver | [[Glorieta-Cambio-Sentido]] |

### 8. Herido en accidente — Abrigar

| Fuente | Lo que dice |
|--------|-------------|
| **Tests simplificados** | "Siempre abrigar al herido" |
| **DGT / Primeros auxilios** | Abrigar **solo si hay shock o frio**. **NO** si hay golpe de calor o quemaduras |
| **Conclusion** | Evaluar antes de actuar |
| → Ver | [[Herido-Accidente-Abrigar]] |

## Discrepancias de Moto (A2)

### 9. A2 — Tractores

| Fuente | Lo que dice |
|--------|-------------|
| **Tests confusos** | "Con A2 se pueden conducir tractores" |
| **DGT** | A2 = motocicletas hasta 35 kW. **Tractores = NO** (requieren B o permiso especifico) |
| **Conclusion** | A2 no incluye maquinaria agricola |
| → Ver | [[A2-No-Tractores]] |

### 10. Carter y lubricacion

| Fuente | Lo que dice |
|--------|-------------|
| **Tests simplificados** | "El carter lleva la lubricacion" |
| **DGT / Mecanica** | El carter es el **deposito**. La **bomba de aceite** lleva la lubricacion |
| **Conclusion** | No confundir deposito con sistema de bombeo |
| → Ver | [[A2-Carter-Lubricacion]] |

---

## Regla de oro

> **Si un test o app de autoescola dice algo diferente de la DGT, prevalece la DGT.**

Muchas discrepancias vienen de:
- Tests **desactualizados** (anteriores a la reforma 2026)
- Preguntas **mal redactadas** (confunden "camion" con "furgoneta")
- Respuestas **simplificadas** que omiten matices (abrigar siempre, depresion del motor siempre)
- **Criterios antiguos** que ya no estan vigentes (exencion de cinturon para repartidores)
