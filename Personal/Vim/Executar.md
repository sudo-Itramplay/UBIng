# Executar

## Run / Build (`<leader>r`)

| Vull... | Key | Mode |
|---------|-----|------|
| Executar fitxer actual | `<leader>rr` | n |
| LaTeX compile | `<leader>rb` | n |
| LaTeX view PDF | `<leader>rv` | n |
| LaTeX stop compile | `<leader>rs` | n |
| LaTeX clean aux files | `<leader>rc` | n |
| LaTeX errors | `<leader>re` | n |

### Filetypes suportats per `<leader>rr`

| Filetype | Comando |
|----------|---------|
| Python | `python3` |
| Lua | `lua` |
| Bash/Shell | `bash` |
| JavaScript | `node` |
| TypeScript | `node` |
