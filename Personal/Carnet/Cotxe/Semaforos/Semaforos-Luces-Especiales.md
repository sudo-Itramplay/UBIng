# Semaforos: Luces Rojas Intermitentes

## Pregunta de examen (errada)

> Una o dos luces rojas encendidas de manera intermitente en un semaforo es por equipo anti incendios (¿Hay mas asi? ¿Es solo para los bomberos?)

## Respuesta correcta

**FALSO.** Dos luces rojas **intermitentes** en un semaforo indican **paso restringido** (no continuar, via cortada). No es para bomberos ni equipos antiincendios.

## Explicacion detallada

### Semaforo de dos focos rojos intermitentes

Este tipo de semaforo:
- **Significado:** via cortada, prohibido continuar
- **Peligro:** posiblemente hay un obstaculo o peligro en la via (obras, accidente, etc.)
- **Accion:** detenerse y esperar a que se apague o se de paso alternativo

### ¿Puede estar relacionado con emergencias?

Segun la normativa vigente, estos semaforos pueden ser activados por:
- **Control de trafico** por obras, accidentes o cortes de via
- **Vehiculos de emergencia** (bomberos, policia, ambulancias) para cortar el paso
- **Paso a nivel** o entradas a tuneles

Sin embargo, **el significado siempre es el mismo:** via cortada, detenerse.

### Otros semaforos especiales

| Tipo | Significado |
|------|-------------|
| **Luz ambar intermitente** | Hay que ceder el paso, maxima precaucion |
| **Semaforo de prioridad** (flechas) | Prioridad en una direccion |
| **Semaforo de peatones** | Paso permitido/parado segun luz |
| **Semaforo de carril** | Carril abierto/cerrado |
| **Semaforo de trafico lento** | Vehiculo lento en la via |

### Errores habituales

| Error | Peligro |
|-------|---------|
| Confundir rojo intermitente con alerta exclusiva de bomberos | Continuar cuando esta prohibido |
| Creer que indica equipo de emergencia en la via | No tiene que ver con presencia de emergencias, sino con corte de via |
| No detenerse en rojo intermitente | Accidente o multa |

## Recordatorio

**Rojo intermitente = VIA CORTADA.** Detenerse y esperar. Puede estar activado por emergencias, obras o control de trafico, pero el significado siempre es el mismo.
