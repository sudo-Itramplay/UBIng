# Git

| Vull... | Key | Mode |
|---------|-----|------|
| Obrir LazyGit | `<leader>gg` | n |
| LazyGit (cwd) | `<leader>gG` | n |
| Git blame | `<leader>gb` | n |
| Git browse (GitHub) | `<leader>gB` | n |
| File history | `<leader>gf` | n |
| Git log | `<leader>gL` | n |
| Git log (fitxer) | `<leader>gl` | n |
| Git status | `<leader>gs` | n |
