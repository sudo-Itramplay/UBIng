La **World Wide Web (WWW)** és el sistema distribuït més conegut: servidors HTTP (web servers) i clients (browsers). És una aplicació [[Model Client-Servidor|client-servidor]] orientada a la connexió sobre **HTTP** (port 80 per defecte).

**Tres elements clau:**
- **URL/URI:** sistema de noms i adreces. `protocol://user:pass@host:port/path?param=value`.
- **[[HTTP]]:** protocol de transmissió.
- **HTML:** llenguatge de marcatge (basat en SGML, definit pel W3C).

Conceptes: [[HTTP]] · [[Contingut Dinàmic Web]] · [[MVC]].

Relacionat: [[HTTP]] · [[Model Client-Servidor]] · [[Web Services]]
