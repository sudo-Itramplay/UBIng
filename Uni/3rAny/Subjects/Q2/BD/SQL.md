El **SQL (Structured Query Language)** és el llenguatge estàndard de l'ANSI per operar amb servidors de [[Base de Dades]] relacionals. Incorpora nativament l'[[Àlgebra Relacional]] ("la versió amigable").

**Referència: Diapositives 4-6**

- És **declaratiu:** especifiquem _què_ volem, no _com_ obtenir-ho.
- Redueix els problemes de migració entre productes [[DBMS]].
- **Evolució:** SEQUEL a IBM → SQL-86, SQL-92, SQL:1999 → versions modernes.

**Estructura MySQL:** sistema **client-servidor**; un servidor allotja múltiples **bases de dades** (esquemes), cadascuna amb **taules**. MySQL Workbench és l'IDE oficial.
