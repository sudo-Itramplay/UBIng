# Autovia vs Autopista: Diferencias Clave

## Pregunta de examen (errada)

> Como identificar si es autovia o autopista, cual es la diferencia?

## Respuesta correcta

**VERDAD parcial.** Autopista y autovia son vias rapidas, pero tienen **diferencias importantes** en accesos, arcen, pasos a nivel y senalizacion.

## Explicacion detallada

### Cuadro comparativo

| Caracteristica | Autopista | Autovia |
|----------------|-----------|---------|
| **Denominacion** | AP-XX o A-XX | A-XX o N-XXX (via rapida) |
| **Accesos** | **Controlados** (solo por intercambiadores) | **Limitados** (pueden tener algun cruce) |
| **Pasos a nivel** | **NO tiene** | **Puede tener** pasos a nivel o cruces |
| **Arcen** | Siempre **pavimentado** y ancho | Puede ser mas **estrecho** o no pavimentado |
| **Intersecciones** | A distinto nivel (puentes) | A distinto nivel, pero puede tener algun cruce |
| **Velocidad maxima** | 120 km/h | 100 km/h |
| **Carril de aceleracion/desaceleracion** | Siempre | A veces mas cortos |
| **Bermas (separador central)** | Si, anchas y protegidas | Pueden ser mas estrechas |
| **Peaje** | Puede ser de peaje | Generalmente gratuita |

### Diferencias clave

#### 1. Accesos
- **Autopista:** los accesos son **exclusivos** por intercambiadores (enlaces). No se puede entrar ni salir libremente.
- **Autovia:** los accesos son **limitados**, pero puede haber algun cruce o entrada directa.

#### 2. Pasos a nivel
- **Autopista:** **NUNCA** tiene pasos a nivel ni cruces con otras vias.
- **Autovia:** **PUEDE** tener pasos a nivel o cruces (aunque es raro).

#### 3. Arcen
- **Autopista:** el arcen es **siempre pavimentado** y tiene un ancho suficiente.
- **Autovia:** el arcen puede ser **mas estrecho** o incluso no estar pavimentado.

#### 4. Velocidad maxima
- **Autopista:** 120 km/h (turismos)
- **Autovia:** 100 km/h (turismos)

### Como identificarlas en la practica

| Elemento | Autopista | Autovia |
|----------|-----------|---------|
| **Senal de inicio** | Senal azul con "Autopista" y simbolo | Senal azul con "Autovia" y simbolo |
| **Poste kilometro** | Fondo **azul** | Fondo **verde** |
| **Peaje** | A veces cabines de peaje | No hay peaje |
| **Cruces** | Solo por enlaces (puentes) | Puede haber cruces |

### Errores habituales

| Error | Peligro |
|-------|---------|
| Creer que autovia = 120 km/h | La velocidad maxima es 100 km/h |
| Confundir autopista con autovia | Multa por exceso de velocidad |
| No saber que la autovia puede tener pasos a nivel | No anticipar el peligro |
| Pensar que todas las vias rapidas son autopistas | La autovia tiene menos requisitos |

## Recordatorio

**Autopista = acceso controlado, 120 km/h, arcen ancho.** **Autovia = acceso limitado, 100 km/h, puede tener cruces.**
