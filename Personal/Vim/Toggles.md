# Toggles

> Prefix: `<leader>u`

| Vull... | Key |
|---------|-----|
| Toggle inlay hints | `<leader>uh` |
| Toggle wrap | `<leader>uw` |
| Toggle spellcheck | `<leader>us` |
| Toggle line numbers | `<leader>ul` |
| Toggle diagnostics | `<leader>ud` |
| Toggle conceal | `<leader>uc` |
| Toggle format on save | `<leader>uf` |
| Toggle signcolumn | `<leader>ug` |
| Toggle statusline | `<leader>uL` |
| Toggle tabline | `<leader>uT` |

## Configuració personalitzada

| Opció | Valor |
|-------|-------|
| Colorscheme | kanagawa-dragon |
| Números relatius | Activat |
| Scroll animat (Snacks) | Desactivat |
| Notícies LazyVim | Desactivat |
| Transparència | Activat (bg transparent) |
