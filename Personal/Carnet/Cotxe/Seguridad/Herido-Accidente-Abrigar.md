# Herido en Accidente: ¿Abrigar Siempre?

## Pregunta de examen (errada)

> Siempre se tiene que abrigar a un herido de accidente.

## Respuesta correcta

**FALSO.** No siempre se debe abrigar a un herido. Depende de su estado: si tiene **signos de shock** (hipotermia, palidez, sudoracion fria), si se le abriga; pero si tiene **golpe de calor** o **hemorragia externa con sangre caliente**, no se le debe abrigar.

## Explicacion detallada

### Regla general: mantener la temperatura

El objetivo es que el herido **no pierda ni gane temperatura** de forma excesiva. Esto depende de:
- La temperatura ambiental
- El estado del herido (shock, golpe de calor, quemaduras)
- El tipo de lesiones (hemorragias, fracturas, quemaduras)

### ¿Cuando ABRIGAR?

- **Shock (hipovolemico):** el herido esta palido, sudor frio, pulso debil, temblor
- **Hipotermia:** temperatura corporal baja
- **Ambiente frio:** para evitar que pierda calor
- **Inmovilizado:** al no moverse, genera menos calor

### ¿Cuando NO ABRIGAR?

- **Golpe de calor:** temperatura alta, piel roja y seca, confusion
- **Quemaduras extensas:** la piel ya no regula la temperatura
- **Hemorragia externa con sangre caliente:** se debe enfriar la zona (compresion con agua fria)
- **Ambiente caluroso:** abrigar empeoraria la situacion

### Acciones segun el estado del herido

| Estado del herido | ¿Abrigar? | Accion |
|-------------------|-----------|--------|
| **Shock (hipovolemico)** | **SI** | Cubrir con manta, mantener calor |
| **Hipotermia** | **SI** | Cubrir, evitar perdida de calor |
| **Golpe de calor** | **NO** | Desabrigar, ventilar, agua fresca |
| **Quemaduras graves** | **NO** | No cubrir con tela, usar aposito esteril |
| **Hemorragia externa** | Parcial | Cubrir al herido pero enfriar la herida |

### Errores habituales

| Error | Peligro |
|-------|---------|
| Abrigar siempre a todo herido | Puede empeorar un golpe de calor o quemaduras |
| No abrigar a un herido en shock | El shock se agrava con la hipotermia |
| Cubrir heridas sin limpiar | Infeccion |
| Mover al herido para abrigarlo | Agravar lesiones de columna |

## Recordatorio

**Abrigar solo si hay shock o frio.** No abrigar si hay golpe de calor o quemaduras. La prioridad es evaluar el estado antes de actuar.
